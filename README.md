# url-shortener
url shortener written in nodejs

## Install
```
git clone https://github.com/simba-fs/url-shortener.git
cd url-shortener
npm i 
cat DB=/path/to/your/db >> .env
cat BASEURL=/your/domain/name
npm start
```
